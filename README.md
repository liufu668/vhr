# vhr
微人事是一个前后端分离的人力资源管理系统！
## 后端技术栈
1.Spring Boot

2.Spring Security

3.MyBatis

4.MySQL

5.Redis

6.RabbitMQ

7.Spring Cache

8.WebSocket

## 前端技术栈
1.Vue

2.ElementUI

3.axios

4.vue-router

5.Vuex

6.WebSocket

7.vue-cli4

## 项目基础
1、新建一个vhr数据库

2、下载并配置好Redis和RabbitMQ，下载到Windows系统或Linux系统都可以，看哪个方便

3、此处只专注于后端代码的实现，前端直接复制、粘贴代码，部署好就完了

## 前端部署
